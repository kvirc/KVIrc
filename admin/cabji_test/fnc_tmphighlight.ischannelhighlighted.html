$tmphighlight.isChannelHighlighted
Returns 1 if the channel is highlighted on this IRC network, and 0 otherwise
Usage
<boolean> $tmphighlight.isChannelHighlighted[window_id:integer]
Description
This command returns 1 if the channel is highlighted on this IRC network, and 0 otherwise.
