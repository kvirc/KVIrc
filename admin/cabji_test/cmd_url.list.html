url.list
Opens URL list
Usage
url.list
Description
This command opens a window containing the URLs' list. In the list there is other information:
Window: window where the URL has been shown last
Count: number of URLs shown
Timestamp: date/time when the URL has been shown first

Clicking right on the URL column of the list a menu will popup, through it you can remove the selected item, find the URL in the window it appeared last, and say it to: @Console, Channels, Queries, DCC Chats and User windows.
The list is saved to file when you click on the menu item or when you unload the plugin on condition that you have checked the relative checkbox in configuration dialog.

You can also open the URL in your web browser double clicking on it in the URL list window.
