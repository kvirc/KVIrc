$mediaplayer.localFile
Returns the name of the local file played by the media player
Usage
$mediaplayer.localFile()
Description
Returns the name of the local file played by the currently selected media player interface. This function returns an empty string if the player is not playing, there is a communication error or the media played is not a local file (e.g. a stream or another kind of media).
Take a look at the mediaplayer module documentation for more details about how it works.
