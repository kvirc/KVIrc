whowas
Requests user information
Usage
whowas <nickname>
Description
Requests information about an IRC user specified by <nickname>.
WHOWAS requests history records. See RFC2812 for more info.
This command is connection dependent.
Examples
 


# Query past info about Pragma

whowas Pragma
