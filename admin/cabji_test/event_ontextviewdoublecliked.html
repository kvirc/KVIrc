OnTextViewDoubleClicked
Text view link double click customization
Parameters
 
$0 = Actual name
$1 = Visible name
$2 = Command name
Window
The window that the user has clicked in
Description
Actual name is equal to the command name if command name is not null. Visible name is the link text (as it shown by user). Command name is set if the visible name differs with the actual link target (e.g. probably contains mIRC colors). It is equal to the actual target name. This event is triggered when a user has double clicked on a text view and has not hit any link.
I have no idea what you can do in this event, maybe some sort of channel-central dialog? :)
