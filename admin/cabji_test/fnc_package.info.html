$package.info
Returns information about a KVIrc package file
Usage
<hash> $package.info(<package_path:string>)
Description
Returns a set of information about a KVIrc package file (*.kvt or *.kva). The set is a structured hash. If the specified path does not lead to a valid KVIrc package the returned hash will contain a single element with key "error".
