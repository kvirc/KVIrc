OnMeChanOwner
Someone has given the +q flag to the local user
Parameters
 
$0 = Source nickname
$1 = Source username
$2 = Source hostname
Window
Channels window
Description
Triggered when someone sets a +q (has made Channel owner) flag on the local user in the active channel.
