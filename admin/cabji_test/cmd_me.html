me
Sends a CTCP ACTION
Usage
me <text:string>
Description
Sends a CTCP ACTION to the current channel, query or DCC chat.
If you execute it in any other window type, you will get an error.
If you want to use this command in a window that is not a channel query or DCC chat, you may use the standard -r switch.
Examples
 


me is Hungry!
