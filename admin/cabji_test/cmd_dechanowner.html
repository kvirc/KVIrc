dechanowner
Removes chan owner status from the specified users
Usage
dechanowner <nickname_list>
Description
Removes channel owner status to the users specified in <nickname_list>, which is a comma separated list of nicknames. This command works only if executed in a channel window. The command is translated to a set of MODE messages containing a variable number of -q flags. This command is connection dependent.
Examples
 


dechanowner Pragma,Crocodile
