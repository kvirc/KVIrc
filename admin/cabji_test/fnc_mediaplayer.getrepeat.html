$mediaplayer.getRepeat
Return if Repeat is on.
Usage
$mediaplayer.getRepeat()
Description
Return the value of the Repeat flag for the current track (1 for ON, 0 for OFF).
Take a look at the mediaplayer module documentation for more details about how it works.
