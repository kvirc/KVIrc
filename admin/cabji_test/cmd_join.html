join
Joins the specified channels
Usage
join <channel list> [key list]
Description
Joins the channels specified in <channel list> by using the optional keys in the [key list].
This command is "server based"; you will see its results only after the server acknowledges it.
This command is connection dependent.
Examples
 


join #kvirc

join #linux-it,#xmms,#kde linus
