mode
Sends a MODE IRC message
Usage
mode <target> <modeflags> [mode parameters]
Description
Sends a MODE IRC message to the server of the current IRC context.
The parameters are not modified in any way by KVIrc: so you should use the RFC1459 syntax.
This command is connection dependent.
Examples
 


mode #kvirc +oo-b Pragma Buti *!*root@*
